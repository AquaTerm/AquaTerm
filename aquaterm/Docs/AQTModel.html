Class Description

A class representing a collection of objects making up the plot. The objects can be leaf object like GPTPath and GPTLabel or a collection itself (not exploited at present).

Instance Variables

NSMutableArray *modelObjects;
AQTColorMap *modelColorMap;

modelObjectsAn array of GPTGraphic objects (leaf or collection)
modelColorMapA model-wide mapping of colors so that the (single) property inspector knows the colormap of each window (model)
