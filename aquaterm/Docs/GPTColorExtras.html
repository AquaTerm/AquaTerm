NSColor (GPTColorExtras)


Declared In:
GPTColorExtras.h


Category Description

A category to provide some useful color mappings.

Method Types

+ interpolateColorFrom:to:by:
+ getColorFromIndex:

Class Methods

getColorFromIndex:

+ (NSColor *)getColorFromIndex:(int)param

Gnuplot uses is a color index to map linestyles to a set of fixed colors. The index is taken modulo max_number_of_colors. Negative numbers have special meanings (-2 = axes, -1 = grid).

interpolateColorFrom:to:by:

+ (NSColor *)interpolateColorFrom:(NSColor *)c0 to:(NSColor *)c1 by:(double)param

Gnuplot uses is a mapping from [0 1] to an arbitrary color space and +(NSColor *)interpolateColorFrom: to: by: defined does that for us. It returns an instance of NSColor that is an interpolation between the colors c0 and c1. When gray is 0 the color is c0 and when gray is 1 the returned color is c1.
