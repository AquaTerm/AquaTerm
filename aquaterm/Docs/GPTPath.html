Class Description

A leaf object class representing an actual item in the plot. Since the app is a viewer we do three things with the object: create (once), draw (any number of times) and (eventually) dispose of it.

Instance Variables

NSBezierPath *_gptPath;
NSColor *_strokeColor;
NSColor *_fillColor;
int _linestyle;
BOOL _isFilled;
float _gray;
float _linewidth;

_gptPathA collection of bezier paths sharing the same (style)properties
_strokeColorThe (actual) stroke- and textcolor for a graphic object
_fillColorThe fill color for a bezier path
_linestyleThe linestyle given by gnuplot, could mean color or dash depending on graphic object
_isFilledTRUE => fill and stroke in _fillColor; FALSE => stroke in _strokeColor
_grayThe parametrized fill color [0..1]
_linewidthThe linewidth for a bezier path ** NOT USED **
