Class Description

GPTReceiverObject is the class controlling the DO connection. It creates an NSConnection and register the service "aquatermServer" with the system.

gptConnectionThe DO connection object
listenerThe object that listens to the messages from this object
aqtModelThe graph being built
currentFigureThe current term number, set by "set term aqua <n>" in gnuplot
currentFontThe current font, defaults to Times-Roman at 16pt
