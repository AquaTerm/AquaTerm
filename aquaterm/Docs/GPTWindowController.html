Class Description

GPTWindowController is a subclass of NSWindowController that controls an AquaTerm window. Every instance of a GPTWindowController holds a reference to the rendering GPTView and to the GPTModel received from gnuplot.

Instance Variables

IBOutlet id viewOutlet;
GPTModel *model;
NSString *viewTitle;
int viewIndex;

viewOutletPoints to the rendering view
modelHolds the model for the view
viewTitleThe title of the model from gnuplot's set term aqua <n> title command
viewIndexThe number by which gnuplot refers to the model
