The docs that used to live here has moved!

All docs are available from http://aquaterm.sourceforge.net

Class docs are created on the fly by selecting target Docs in PB and clicking build.
The can be found in directory aquaterm/build/docs.

Note. You need autodoc (http://softrak.stepwise.com/category?os=20&cat=3184) and you will have to change the path to makedocs script in PB buid settings.
