setModel:forView:

- (void)setModel:(AQTModel *)gptModel forView:(unsigned)index

When the controller receives a model from the sender (e.g. gnuplot) this method hands it over to the windowController corresponding to the current terminal (figure), indexed from zero and upwards. If a windowController for terminal <n> doesn't exist, it is created.
